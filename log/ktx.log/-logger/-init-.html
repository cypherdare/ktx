Logger(tag: String, debugPrefix: String = "[DEBUG] ", infoPrefix: String = "[INFO] ", errorPrefix: String = "[ERROR] ")

A simple logging utility class which caches its tags.

Parameters

tag - name of the logger included in log tags. Proceeded with a prefix of the specified log type.

debugPrefix - will proceed tag in debug logs.

infoPrefix - will proceed tag in info logs.

errorPrefix - will proceed tag in error logs.
