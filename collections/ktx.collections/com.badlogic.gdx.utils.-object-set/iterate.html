iterate

inline fun <Type> GdxSet<Type>.iterate(action: (Type, MutableIterator<Type>) -> Unit): Unit

Allows to iterate over the array with access to MutableIterator, which allows to remove elements from the collection during iteration.

Parameters

action - will be invoked for each set element. Allows to remove elements during iteration. The first function argument is the element from the set, the second is the set iterator. The iterator is guaranteed to be the same instance during one iteration.
