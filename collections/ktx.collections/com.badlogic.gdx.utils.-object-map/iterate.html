iterate

inline fun <Key, Value> GdxMap<Key, Value>.iterate(action: (Key, Value, MutableIterator<Entry<Key, Value>>) -> Unit): Unit

Allows to iterate over the map with Kotlin lambda syntax and direct access to MutableIterator, which can remove elements during iteration.

Parameters

action - will be invoked on each key and value pair. Passed iterator is ensured to be the same instance throughout the iteration. It can be used to remove elements.
