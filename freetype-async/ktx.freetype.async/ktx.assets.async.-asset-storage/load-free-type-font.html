loadFreeTypeFont

suspend inline fun AssetStorage.loadFreeTypeFont(path: String, setup: FreeTypeFontParameter.() -> Unit = {}): BitmapFont

Allows to customize parameters of a loaded FreeType font.

path is the file path to the FreeType font file. Must be compatible with the AssetStorage.fileResolver.

setup can be used to specify and customize the parameters of the loaded font. It will be inlined and invoked on a FreeTypeFontParameter.

Returns the result of font loading. See AssetStorage.load for lists of possible outcomes.

Note that you can also call AssetStorage.loadSync or AssetStorage.loadAsync directly if needed, but you must pass FreeTypeFontParameter. See freeTypeFontParameters utility.
