tab

inline fun KTabbedPane.tab(title: String, savable: Boolean = false, closeableByUser: Boolean = true, init: KTab.() -> Unit = {}): Tab

Begins creation of new Tab using type-safe builder. Newly created tab will be added to tabbed pane automatically so there is no need to call add manually.

Parameters

title - title of the Tab.

savable - see Tab.savable.

closeableByUser - see Tab.closeableByUser.

init - allows to customize the Tab.

Return
a new Tab instance added to this pane.
