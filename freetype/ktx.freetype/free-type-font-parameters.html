freeTypeFontParameters

inline fun freeTypeFontParameters(file: String, setup: FreeTypeFontParameter.() -> Unit = {}): FreeTypeFontLoaderParameter

Syntax sugar for FreeTypeFontLoaderParameter initialization. Used internally by loadFreeTypeFont.

Parameters

file - path to the font file. Must be the same as the path passed to the loader.

setup - should specify font parameters. Will be invoked on a new instance of FreeTypeFontParameter. Inlined.
