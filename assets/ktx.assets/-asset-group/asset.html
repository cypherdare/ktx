asset

protected fun <reified T : Any> asset(fileName: String, params: AssetLoaderParameters<T>? = null): Asset<T>

Creates a delegate for an asset, queues it for loading, and registers it as a member of this group. Beware that calling loadAll might increment the AssetManager's reference count for this asset an additional time.

Parameters

fileName - Name of the file used to reference this asset by the AssetManager. This value will be prefixed with filePrefix.

params - optional asset loading parameters which might affect how the assets are loaded. Can be null.

Return
an Asset, registered as a member of this AssetGroup.
