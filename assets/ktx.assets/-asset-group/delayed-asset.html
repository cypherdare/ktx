delayedAsset

protected fun <reified T : Any> delayedAsset(fileName: String, params: AssetLoaderParameters<T>? = null): Asset<T>

Creates a delegate for an asset and registers it as a member of this group. It is not queued for loading. Beware that if the asset property (or delegated property) of the returned Asset is accessed before it is queued for loading, it will eagerly be loaded by the AssetManager. This may cause loadAll to throw off AssetManager's reference counts.

Parameters

fileName - Name of the file used to reference this asset by the AssetManager. This value will be prefixed with filePrefix.

params - optional asset loading parameters which might affect how the assets are loaded. Can be null.

Return
an Asset, registered as a member of this AssetGroup.
