weldJointWith

inline fun Body.weldJointWith(body: Body, init: WeldJointDef.() -> Unit = {}): WeldJoint

Allows to create a WeldJoint. this Body will be set as the JointDef.bodyA and will be available through Joint.getBodyA.

Parameters

body - will be set as JointDef.bodyB and available through Joint.getBodyB.

init - inlined. JointDef.bodyA, JointDef.bodyB and JointDef.type must not be changed. Invoked after joint definition default values are preset.

Return
a new instance of WeldJoint created with the customized WeldJointDef.
