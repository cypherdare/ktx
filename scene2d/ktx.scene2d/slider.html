slider

inline fun <S> KWidget<S>.slider(min: Float = 0f, max: Float = 1f, step: Float = 0.01f, vertical: Boolean = false, style: String = if (vertical) defaultVerticalStyle else defaultHorizontalStyle, skin: Skin = Scene2DSkin.defaultSkin, init: Slider.(S) -> Unit = {}): Slider

Parameters

min - minimum value displayed by the slider. Defaults to 0.

min - maximum value displayed by the slider. Defaults to 1.

step - the size of a single step between two values. Defaults to 0.01.

vertical - true if the widget is vertical, false if horizontal.

style - name of the widget style. Defaults to defaultVerticalStyle if the widget is vertical or defaultHorizontalStyle if the widget is horizontal.

skin - Skin instance that contains the widget style. Defaults to Scene2DSkin.defaultSkin.

init - will be invoked with the widget as "this". Consumes actor container (usually a Cell or Node) that contains the widget. Might consume the actor itself if this group does not keep actors in dedicated containers. Inlined.

Return
a Slider instance added to this group.
