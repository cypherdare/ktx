open fun <T : Actor> T.node(icon: Drawable? = null, expanded: Boolean? = null, selectable: Boolean? = null, userObject: Any? = null): T

Allows to customize properties of the Node storing this actor .

Parameters

icon - will be drawn next to the actor.

expanded - true to expand the node, false to hide its children.

selectable - if false, this node cannot be selected.

userObject - custom object assigned to the node.

Return
this actor.
