fun <reified Type : Any> bindSingleton(singleton: Type): Unit

Allows to bind a singleton to the chosen class.

Parameters

singleton - will be converted to a provider that always returns the same instance. If no type argument is passed, it will be bind to its own class.

Exceptions

InjectionException - if provider for the selected type is already defined.

Allows to bind a singleton instance to multiple classes in its hierarchy.

Parameters
