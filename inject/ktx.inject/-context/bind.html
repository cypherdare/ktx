bind

inline fun <reified Type : Any> bind(noinline provider: () -> Type): Unit

Allows to bind a provider producing instances of the selected type.

Parameters

provider - will be bind with the selected type. If no type argument is passed, it will be bind to the same exact class as the object it provides.

Exceptions

InjectionException - if provider for the selected type is already defined.

fun <Type : Any> bind(vararg to: KClass<out Type>, provider: () -> Type): Unit

Allows to bind a provider to multiple classes in hierarchy of the provided instances class.

Parameters

to - list of interfaces and classes in the class hierarchy of the objects provided by the provider. Any time any of the passed classes will be requested for injection, the selected provider will be invoked.

provider - provides instances of classes compatible with the passed types.

Exceptions

InjectionException - if provider for any of the selected types is already defined.
